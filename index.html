Get started in minutes

  1. Clone the repo and install dependencies.
  2. Run `python app.py` for the dashboard or `bash vibecheck.sh` for CLI.
  3. Ship the report or integrate into CI.
git clone https://github.com/TheVibeChecker/TheVibeChecker.github.io.git
cd TheVibeChecker.github.io
python app.py
